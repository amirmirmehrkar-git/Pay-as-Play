## üí∞ Wallet Management

### Viewing Your Balance

Your wallet balance is displayed in the top navigation and on the wallet page.

**Balance Status:**
- **Green:** Sufficient balance
- **Yellow:** Low balance warning
- **Red:** Critical balance

### Adding Funds (Top-up)

1. Go to Wallet page
2. Click "Add Funds" or "Top-up"
3. Enter amount
4. Select payment method
5. Complete payment
6. Funds added to wallet

### Auto Top-up

Set up automatic top-up to never run out of balance:

1. Go to Settings ‚Üí Wallet ‚Üí Auto Top-up
2. Enable auto top-up
3. Set threshold amount
4. Set top-up amount
5. Select payment method
6. Confirm with 2FA
