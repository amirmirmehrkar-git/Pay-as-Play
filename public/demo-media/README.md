# Demo Media Files

این پوشه برای فایل‌های رسانه نمونه (ویدیو و صوت) استفاده می‌شود.

## ساختار پیشنهادی:

```
public/demo-media/
├── video/
│   ├── sample-video.mp4
│   └── ...
├── audio/
│   ├── sample-audio.mp3
│   └── ...
└── README.md
```

## فرمت‌های پشتیبانی شده:

- **ویدیو:** MP4, WebM, OGG
- **صوت:** MP3, WAV, OGG

## نحوه استفاده:

1. فایل‌های ویدیو را در پوشه `video/` قرار دهید
2. فایل‌های صوت را در پوشه `audio/` قرار دهید
3. فایل‌ها به صورت خودکار در سیستم استفاده می‌شوند

## مثال:

- `/demo-media/video/sample-video.mp4`
- `/demo-media/audio/sample-audio.mp3`
